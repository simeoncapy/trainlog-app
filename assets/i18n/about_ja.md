# Trainlogとは？

Trainlogは、あなたの移動履歴を記録・管理するためのオンラインサービスです。電車、バス、フェリー、飛行機での移動を地図上でわかりやすく確認できます。いつ・どこへ・どのように移動したかの統計情報も確認できます。よく利用する駅や事業者、さらには自国の鉄道ネットワークのどの程度を利用したかといった有益なデータも取得できます。

また、旅行の計画を共有できるリンクを作成し、誰とでも共有することができます。さらに、Trainlogコミュニティ内で他のユーザーと比較することも可能です。

# 画像に関する情報

事業者のロゴは説明目的のためにのみ使用されており、各事業者による承認を意味するものではありません。ロゴの削除を希望する事業者は、以下のメールアドレスまでご連絡ください：[admin@trainlog.me](admin@trainlog.me)。

# Trainlogの運営について

Trainlogは**Boreal Baguette Studios**によって運営されており、以下のウェブサイトからアクセスできます：[https://trainlog.me/](https://trainlog.me/)。

このウェブサイトはオープンソースプロジェクト（次のセクション参照）であり、主に**Boreal Baguette Studios**によって開発されています。

スマートフォンアプリもオープンソースプロジェクトであり、主に[scapy](https://cv.scapy.fr)によって開発されています。
